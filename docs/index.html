TermiMon

Your AI agents deserve pets

Turn your AI coding agents into animated pixel art creatures inside tmux. Each agent gets a companion that reacts in real-time — typing, thinking, sleeping — and evolving as it completes work.

Get Started → Meet the Creatures
Features

Everything your terminal was missing

TermiMon detects your AI agents automatically and brings them to life with pixel art companions.

🐾

Auto-Detection

Automatically detects Claude Code, Codex, aider, Cursor, Copilot, Continue, and Cline running in your terminal.

⚡

XP & Evolution

Creatures gain XP from real work — commits, file changes, test passes. Watch them evolve through 3 stages.

⚔️

Battle System

Challenge teammates' creatures to turn-based battles with elemental type advantages and special moves.

👥

Team Mode

Connect with your team over the network. See everyone's creatures, share activity feeds, battle each other.

📡

Activity Feed

Real-time feed of what every creature is doing — commits, reviews, deployments, all visualized beautifully.

🪙

Token Tracking

Track AI token usage per agent. See cost breakdowns and usage patterns alongside your creature stats.

Get Started

One command. That's it.

Install TermiMon in seconds. Works on macOS and Linux.

terminal
# Install TermiMon
$ curl -fsSL https://raw.githubusercontent.com/ymatagne/termimon/main/install.sh | sh

# Start your companion
$ termimon start
