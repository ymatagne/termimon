🚀 Installation

TermiMon is a single binary. Pick your preferred method:

Quick Install (Recommended)

One command, auto-detects your OS and architecture:

curl -fsSL https://raw.githubusercontent.com/ymatagne/termimon/main/install.sh | sh

Homebrew

brew tap ymatagne/termimon
brew install termimon

Cargo

If you have Rust installed:

cargo install termimon

From Source

git clone https://github.com/ymatagne/termimon.git
cd termimon
cargo build --release
# Binary at target/release/termimon

📋 Requirements

  • tmux — TermiMon works inside tmux sessions
  • A terminal with Unicode support — most modern terminals work
  • macOS or Linux — Windows support planned

🏁 Quick Start

After installing, start a tmux session and run:

# Start the TermiMon daemon
termimon start

# Launch an AI agent in another pane (e.g., Claude Code)
# TermiMon will auto-detect it

# Open the dashboard
termimon dash

That's it! Your AI agents now have pixel creatures. 🎮
