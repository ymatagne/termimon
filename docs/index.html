🎮 TermiMon

Your AI coding agents deserve pets

TermiMon turns your AI coding agents into animated pixel art creatures inside tmux. Each agent gets its own creature that reacts in real-time — typing when writing code, thinking when planning, sleeping when idle, and evolving as it completes work.

$ curl -fsSL https://raw.githubusercontent.com/ymatagne/termimon/main/install.sh | sh


Get Started Meet the Creatures

✨ Features

🐾

Living Creatures

Each AI agent gets an animated pixel art companion with unique personality

🎨

Pixel Art Rendering

16×16 sprites using half-block Unicode characters (▀▄) — no images needed

⚡

Lightweight

Pure Rust single binary. Less than 1% CPU and 20MB RAM

🔌

Auto-Detection

Finds Claude Code, Codex, aider, Cursor, Copilot, Continue, and Cline automatically

📊

Token & Cost Tracking

Per-agent token costs from Claude JSONL transcripts, plus activity feed

🖥️

Status Bar

Creatures live in your tmux status-right — always visible at a glance

🏆

XP & Evolution

Creatures evolve through 3 stages as your agents complete work

🎬

Animations

Breathing idle, fast typing, dimmed sleeping — creatures feel alive

🔀

Agent Switching

Press Enter in the dashboard to jump to any agent's tmux pane

🏁 Quick Start

$ termimon start      # Start the daemon (inside tmux)
$ termimon status     # Check detected agents
$ termimon dash       # Open interactive dashboard
$ termimon pokedex    # Browse creature collection
