📖 Command Reference

All commands are run via the termimon binary.

CommandDescription
termimon startStart the background daemon. Must be inside a tmux session.
termimon stopStop the running daemon.
termimon statusShow detected agents, their creatures, XP, and resource usage.
termimon dashOpen the interactive dashboard with pixel art, stats, and activity feed.
termimon pokedexView your creature collection and evolution progress.
termimon switchSwitch to a specific agent's tmux pane.

🎮 Dashboard Controls

Inside termimon dash:

KeyAction
↑ / kPrevious agent
↓ / jNext agent
EnterSwitch to selected agent's tmux pane
TabCycle dashboard panels
qQuit dashboard

⚙️ Configuration

Config file: ~/.termimon/config.toml

[general]
poll_interval_ms = 2000    # How often to scan for agents
animation_fps = 4          # Sprite animation speed
theme = "default"          # Color theme

[statusbar]
position = "right"         # tmux status bar position
max_creatures = 5          # Max creatures in status bar
show_xp = false            # Show XP in status bar
show_state = true          # Show animation state name

[creatures.assignments]    # Override auto-assignment
claude = "embercli"
codex = "voltprompt"
aider = "shelloise"
cursor = "rustacean"
copilot = "pythorn"
