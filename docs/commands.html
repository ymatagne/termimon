Commands

Everything you can do with the TermiMon CLI.

🚀 Core Commands

CommandDescription
termimon startStart TermiMon and begin monitoring for AI agents
termimon stopStop all TermiMon processes
termimon statusShow current creature status, XP, and activity
termimon watchOpen the live creature view in your terminal

🐾 Creature Commands

CommandDescription
termimon creatureShow your current creature's stats and evolution progress
termimon creature rename <name>Give your creature a custom name
termimon creaturesList all available creature types

⚔️ Battle & Team

CommandDescription
termimon team join <url>Join a team server
termimon team statusShow connected teammates and their creatures
termimon battle <user>Challenge a teammate to a creature battle
termimon feedView the team activity feed

⚙️ Configuration

CommandDescription
termimon configShow current configuration
termimon config set <key> <val>Update a config value
termimon tokensShow token usage stats for tracked agents
termimon resetReset creature data (⚠️ irreversible)
