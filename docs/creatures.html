🐾 Creature Compendium

Every AI coding agent gets a creature companion. Creatures are auto-assigned based on the agent type, or you can configure them manually.

🏆 Evolution System

Creatures evolve as their agents work. XP is earned from:

  • File writes and reads
  • Shell commands executed
  • Git operations (commits, pushes)
  • Token usage
  • Successful task completions

Evolution Stages

StageXP RequiredChanges
Stage 1 — Base0 XPBase sprite, simple animations
Stage 2 — Evolved100 XPEnhanced sprite, new details, faster animations
Stage 3 — Final500 XPFull sprite with special effects and unique idle animation
